# 🩺 PharmaCare - Premium Virtual Pharmacy & Health Assistant

**PharmaCare** is a premium virtual online pharmacy simulator and health assistant application. It features a beautiful, state-of-the-art **Glassmorphism design aesthetic**, smooth micro-animations, and a highly responsive layout (100% mobile-friendly).

---

## ⚡ Project Highlights
*   **Pure Vanilla Tech Stack:** Developed completely using native **HTML5, CSS3, and JavaScript (ES6+)** without any framework overhead. **No node_modules, no dependencies, no builds, no compilation!** Just double-click `index.html` to run in any browser instantly.
*   **All-in-One SPA Concept:** Experience a seamless Single Page Application layout, ensuring fluid view switching and absolute client-side responsiveness.
*   **Rich Interactive Systems:** Loaded with state-of-the-art diagnostic simulations, order workflows, and interactive user components.

---

## 🚀 10 Global Features Included
1.  **🌐 Bilingual Localization Support:** Seamlessly switch the entire application context, product databases, navigational links, and message logs between **Thai (TH)** and **English (EN)** in a single click.
2.  **🌓 Dark Mode Switcher:** Sleek color variable overriding that shifts standard minty hues into navy carbon and glowing emerald neon styles for comfortable night-time browsing.
3.  **💬 Pharmacist AI Chatbot Simulator:** Consult clinical symptoms or products with a built-in automated pharmacist bot (Pharm. Piyawan) that scans key terms and returns custom answers.
4.  **⚠️ EHR Drug Allergy Warning & Blocking:** Simulate an Electronic Health Record (EHR) allergy profiling. If a user registers allergies (e.g., Paracetamol) and attempts to purchase restricted medicines, the system immediately intercepts, blocks the cart addition, and displays a pulsating diagnostic threat warning.
5.  **🌡️ Interactive Symptom Checker:** Step-by-step triage card selecting major illness symptoms (headaches, throat infections, dry eyes, skin irritation) with clinical analysis and pharmacological product recommendations.
6.  **⚖️ 3-Way Side-by-Side Product Comparison:** Tick comparison checkboxes to view up to 3 medical items side-by-side inside a tabular comparison popup displaying specs, pricing, active ingredients, and precautionary statements.
7.  **🧮 BMI Calculator & Nutrient Advisor:** Determine body mass index index thresholds via height/weight inputs, rendering a clinical meter with tailored vitamin and nutrient suggestions.
8.  **📅 Vaccine Scheduling & barcoded E-Tickets:** Select branch location, vaccine types, calendar dates, and times to issue a fully printable medical appointment coupon with a custom barcode.
9.  **🛍️ Checkout Form & E-Receipt Invoice:** Complete simulated payments (Thai PromptPay QR scan or COD) with discount coupon codes (e.g., `WELCOME100`, `HEALTH20`) to produce a custom-designed print-friendly receipt showing detailed breakdowns, loyalty points, and package tracking links.
10. **📊 Interactive Stepper Timeline & Health Expenses Chart:** Track current delivery parcel steps in real-time and review monthly healthcare transaction budgets via visual chart components.

---

## 📂 Project Structure
*   📄 **`index.html`** - Core SPA architecture, input views, diagnostic widgets, and modal frameworks.
*   🎨 **`style.css`** - Custom Glassmorphic CSS system, night mode styles, printer sheets, and e-ticket layout variables.
*   ⚙️ **`app.js`** - Dynamic product database, translation dictionaries, chatbot NLP triage, LocalStorage sync, and order calculations.

---

## 🛠️ How to Run
1.  **Clone or download** this repository folder to your local machine.
2.  Navigate into the workspace folder and double-click the **`index.html`** file using any modern web browser (Google Chrome, Microsoft Edge, Mozilla Firefox, Apple Safari, etc.).
3.  Enjoy exploring all the premium medical simulations of PharmaCare!
